Задание 2

заголовок;

превью (изображение);
дата создания;
признак публикации;
количество просмотров.
Для работы с блогом реализуйте CRUD для новой модели.

CRUD реализуйте на основе CBV (
ListView
,

,

, 
UpdateView
, 
DeleteView
) Соблюдайте нейминг шаблонов для CBV контроллеров - …_list.html, …_detail.html, …_form.html.
